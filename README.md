# eCrimePortal 🚔

A secure and scalable Crime Management & Reporting System built with **Java, Spring Boot, Spring Security, JWT, and SQL**.  
It enables citizens to file complaints online, track case progress, and provides role-based access for police officers and administrators.

---

## ✨ Features
- **User Authentication & Authorization**  
  Secure login with JWT-based token management and role-based routing.
- **Complaint Registration**  
  Citizens can file complaints with details like crime type, location, date, and description.
- **Case Tracking**  
  Real-time status updates with assigned officer details and remarks.
- **Role-Based Access**  
  - **Citizen**: File and track complaints  
  - **Police Officer**: Manage assigned cases  
  - **Admin**: Oversee system operations
- **Database Integration**  
  Cloud-ready SQL database for persistent storage.
- **RESTful APIs**  
  Production-ready endpoints for frontend integration.

---

## 🛠️ Tech Stack
- **Backend**: Java, Spring Boot, Spring Security, JWT
- **Database**: MySQL 
- **Frontend**: React (planned integration)
- **Tools**: Maven, Git, Postman
